# Google エコシステム活用ガイド: Second Brain Writer

このレポジトリは、Google の AI 技術（Gemini CLI および Jules）を最大限に活用して、知識の整理から記事の執筆までを自動化・効率化するためのスケルトンです。

## 1. 前提条件とセットアップ

### 必要なツール
- **Node.js (v18以上):** Gemini CLI および Jules の動作に必要です。
- **Gemini CLI:** Google の AI と対話するためのメインインターフェース。
- **Jules CLI:** 知識ベースの構造化や高度な整理を支援するインテリジェント・エージェント。

### 認証と設定

Gemini CLI および Jules を利用するには、以下のいずれかの方法で認証を行う必要があります。

#### 推奨：Google サブスクリプションでのログイン
Google One AI プレミアム（Gemini Advanced）などのサブスクリプションをお持ちの場合は、ログインによる認証を推奨します。これにより、高性能なモデルをシームレスに利用できます。

```bash
# Gemini CLI でのログイン
gemini login
```

#### 代替：API キー方式
サブスクリプションをお持ちでない場合は、[Google AI Studio](https://aistudio.google.com/) で API キーを取得し、環境変数に設定してください。

```bash
export GOOGLE_API_KEY="あなたのAPIキー"
```

### Jules の初期設定
[Jules Web Console](https://jules.google.com) にアクセスし、GitHub との連携を完了させてください。このレポジトリでは、Jules は単なるコーディング補助ではなく、**「知識の整理・構造化の提案」を行うパートナー**として機能します。

## 4. Jules による知識の整理と提案 (`/jules` コマンド)

このレポジトリにおいて、Jules は**「あなたの第二の脳を整理し、より良い知識のつながりを提案するエージェント」**としてラッピングされています。

- **スキルの提案:** 「現在の knowledge の内容に基づいて、新しい調査用のカスタムスキルを `.agents/skills/` に提案して」
- **知識の再構造化:** 「`concepts/` 内のメモが散らかってきたので、関連性の高いものをまとめて新しいディレクトリ構造を提案して」
- **ミッシングリンクの発見:** 「今のナレッジベースに足りない視点や、次に調べるべきトピックを分析して」

Jules は、大規模なファイル操作能力を活かして、あなたのナレッジベースを常に最適化された状態に保ちます。

---

## 5. カスタマイズ

`.agents/skills/` に自分専用の命令（スキル）を追加することで、Gemini CLI を自分好みに拡張できます。
