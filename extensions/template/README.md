# Extension Template

## 概要
これは新しい拡張機能（アドオン）を作成するためのテンプレートフォルダです。
特定のユースケース（例：技術ブログ執筆、投資分析、タスク管理など）に合わせて「第二の脳」を特化させたい場合に、このフォルダをコピーして使用します。

## 拡張機能の作り方

1. **フォルダのコピー**: この `template` フォルダを、作りたい機能の名前でコピーします（例: `extensions/investment-analysis`）。
2. **`GEMINI.md` の編集**: そのモードに入ったときのAIの「役割（Role）」や「固有のルール」を定義します。
3. **スキルの追加**: そのタスク特有のカスタムプロンプトやスキルがあれば `skills/` フォルダに配置します。
4. **スクリプトの追加**: 自動化したい作業があれば `scripts/` フォルダに配置します。

## 使い方
ユーザーが特定の作業を行いたいとき、AIに「`extensions/[your-extension]/GEMINI.md` を読み込んで」と指示することで、AIのモードが切り替わります。
