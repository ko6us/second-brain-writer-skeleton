# Second Brain Writer Skeleton

AIエージェント駆動型の「第二の脳」構築と執筆ワークフローのためのスケルトンリポジトリ。
モジュラー（アドオン）構造を採用しており、純粋な知識管理から、特定プラットフォーム向けの執筆や専門的な分析まで、柔軟に拡張可能です。

## 🚀 コンセプト
このリポジトリは、日々のインプット（知識）を蓄積し、そこから自分独自の思想（スタンス）を抽出し、一貫性のあるアウトプット（記事など）を生み出すためのワークスペースです。

AIエージェント（Gemini CLI / Jules等）と協調することを前提に設計されており、インプットからアウトプットまでの「思想のライフサイクル」を回すための構造を提供します。

## 特徴
- **Google エコシステム連携**: Gemini CLI によるサブスクリプション認証や Jules による知識整理・スキル提案に対応。
- **モジュラーアーキテクチャ**: `extensions/` 配下で、note執筆や投資分析など「知能の切り替え」が可能。
- **AIネイティブな構造**: `GEMINI.md` による人格定義と `.agents/skills/` による自動化。

## 📂 ディレクトリ構成
- `knowledge/`: あなたの「第二の脳」。あらゆる知識とスタンスのコア蓄積場所。
- `extensions/`: 特定の用途に特化するためのアドオン（例：note執筆、投資分析）。
- `articles/`: 完成した記事や公開用のドキュメント。
- `.agents/skills/`: 第二の脳を育てるための共通のAIカスタムスキル。
- `scripts/`: 自動化用スクリプト。
- `GEMINI.md`: AIエージェントへの基本指示書（プロジェクトの憲法）。

## 🧩 拡張機能（Extensions）
ルートの `GEMINI.md` は純粋な知識の蓄積と構造化（Core）に特化しています。
特定のタスク（例：note執筆）を行いたい場合は、`extensions/` 配下の専用 `GEMINI.md` を読み込ませることで、AIをそのタスクのプロフェッショナル・モードに切り替えることができます。
