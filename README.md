# Second Brain Writer Skeleton

AIエージェント駆動型の「第二の脳」構築と執筆ワークフローのためのスケルトンリポジトリ。

## 🚀 コンセプト
このリポジトリは、日々のインプット（知識）を蓄積し、そこから自分独自の思想（スタンス）を抽出し、一貫性のあるアウトプット（記事）を生み出すためのワークスペースです。

AIエージェント（Gemini CLI / Antigravity等）と協調することを前提に設計されており、インプットからアウトプットまでの「思想のライフサイクル」を回すための構造と自動化ツールを提供します。

## 📂 ディレクトリ構成
- `knowledge/`: あなたの「第二の脳」。あらゆる知識とスタンスの蓄積場所。
- `articles/`: 執筆中の記事。1記事1フォルダ管理。
- `.agents/skills/`: AIエージェントを強化するカスタムスキル。
- `scripts/`: Gist連携や記事作成を補助する自動化スクリプト。
- `GEMINI.md`: AIエージェントへの指示書（プロジェクトの憲法）。

## 🛠 セットアップ
1. このリポジトリをテンプレートとして使用、またはクローンします。
2. Obsidian等のMarkdownエディタで開き、`knowledge/raw/` にメモを放り込み始めます。
3. AIエージェントを起動し、`GEMINI.md` を読み込ませることで、あなたの執筆パートナーとして機能し始めます。

## 🔄 思想のライフサイクル
1. **蓄積 (Ingest)**: 生のメモを `raw/` に保存。
2. **結晶化 (Distill)**: 共通パターンから「スタンス」を抽出。
3. **執筆 (Write)**: スタンスを元に記事を執筆。AIが整合性をチェック。
4. **循環 (Evolve)**: 執筆で得た発見をスタンスに還元。
